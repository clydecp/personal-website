N-Gram Text Generator
Developed a text generation tool using n-grams to create random sentences based on a given corpus. The project involved text normalization, tokenization, and probabilistic modeling to generate coherent and grammatically correct sentences. Implemented features for handling various n-gram sizes, and addressed punctuation formatting and spacing issues.
Made With: Python, Scikit, Pytorch
